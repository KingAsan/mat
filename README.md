# MathSprint

Веб-приложение для тренировки решения математических задач:
- арифметика
- дроби
- уравнения
- проценты
- смешанный режим

## Возможности

- Выбор темы, сложности и количества задач.
- Таймер на сессию и прогресс по задачам.
- Подсказки и пропуск задач.
- Подсчет очков с учетом скорости, серии правильных ответов и штрафов.
- Статистика тренировки: правильные ответы, ошибки, пропуски, подсказки.
- Сохранение лучшего результата в `localStorage` для каждой комбинации темы и сложности.

## Структура проекта

- `index.html` - интерфейс приложения.
- `styles.css` - стили, адаптивность и анимации.
- `app.js` - логика генерации задач, проверки ответов, очков и таймера.

## Запуск

1. Откройте файл `index.html` в браузере.
2. Выберите тему, сложность и количество задач.
3. Нажмите `Начать тренировку`.

Дополнительно можно запустить через любой локальный статический сервер (по желанию).
